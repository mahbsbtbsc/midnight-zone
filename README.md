# The Midnight Zone

An interactive deep sea bioluminescence visualisation built with p5.js.

## Live Site
[View the visualisation here](https://mahbsbtbsc.github.io/midnight-zone/)

## About
This project visualises bioluminescent deep sea species across ocean depth zones.
Scroll to descend, hover to see creatures react, click any organism to learn facts about it.

## Data Sources
- GBIF API (api.gbif.org) — live species occurrence records queried by taxon key
- OBIS dataset (obis.org) — supplementary species observation counts and region data

## Built With
- p5.js v1.9.3
- GBIF Occurrence API v1
- Vanilla HTML/CSS
