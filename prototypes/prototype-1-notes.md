# Prototype 1 — Vanilla Canvas

First working version. Used plain HTML5 canvas (no p5.js).
Hardcoded species data, no API connection.
All three interactions working: scroll, cursor repulsion, click for facts.

## What worked
- Core scroll-to-descend mechanic
- Bioluminescent glow layers
- Marine snow particles

## What needed changing
- No real data source
- Not built in p5.js (brief requires a library)
- Creatures were just circles — no silhouettes
